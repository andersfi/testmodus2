# Ferskvannsfisk i Trondheim – GBIF-forekomster

An interactive web map showing GBIF occurrence records for freshwater fish species in Trondheim municipality, Norway.

## App

**https://andersfi.github.io/testmodus2/**

## Features

- 14 freshwater fish species with occurrence data from [GBIF](https://www.gbif.org)
- Dropdown species selector with Norwegian, English, and scientific names
- Interactive map with clickable occurrence points linking back to GBIF
- Trondheim municipality boundary overlay

## Data

Occurrence data fetched from the [GBIF Occurrence API](https://www.gbif.org/developer/occurrence) and embedded as static data. Municipality boundary from [Geonorge/Kartverket](https://www.geonorge.no). Data license: CC-BY.
